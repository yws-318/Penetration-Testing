# Happy Hacking 출력

apache의 폴더로 이동   
```
cd /var/www/html
```
</br>

실행시킬 php 내용 확인
```
sudo nano start.php
```
</br>

apache2 실행
```
service apache2 start
```
</br>

웹 브라우저에서 url 입력
```
http://localhost/start.php
```
